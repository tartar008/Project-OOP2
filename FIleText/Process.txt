## มี 2 ไฟล์หลักๆ ที่จะเห็นคือ

    - FileText :
        เป็นโฟเดอร์สำหรับจะวางพวกคำสั่งหรือเขียนอะไรเพิ่มเติมที่เป็น Text ได้
    - Resource :
        เป็นโฟเดอร์ที่ให้สำหรับ code ที่เอามาจากที่อื่นๆเช่น ChatGPT แล้วอยากรัน แต่ไม่อยากรวมเข้ากับ code หลัก ให้รันผ่านโฟลเดอร์นั้นก่อนได้เลย
    -Scriptes :
        เป็นโฟเดอร์ที่ให้สำหรับเก็บ Scriptes ต่างๆที่จะใช้ RUN ทีเดียว รวมถึงการทำ Script Test ด้วย
    -Target :
        เป็นโฟเดอร์ที่เก็บของ Project จริงๆ

    - อาจจะสร้าง JSON ไฟล์ สำหรับเก็บ Statement ของ Room ด้วย
